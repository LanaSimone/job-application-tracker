# Job Application Tracker

A full-stack web application for tracking job applications, application details, and status updates in one place.

## Features
- Add new job applications
- View saved applications
- Edit existing application information
- Update application status
- Delete applications
- Manage data through a simple full-stack interface

## Tech Stack
- React
- Node.js
- Express
- SQLite

## Project Status
This project is currently in progress. The current version includes full CRUD functionality and status management, with additional improvements planned as development continues.

## Why I Built This
I built this project to strengthen my full-stack development skills and create a practical tool for organizing and tracking job applications.
