# Job Application Tracker

A full-stack web application for securely tracking and managing job applications in one place.

Built with a React frontend and a Node.js/Express backend, featuring JWT-based authentication and user-specific data handling.

## Features
- User authentication (register, login, demo account)
- User-specific dashboards (each user only accesses their own data)
- Add, update, and delete job applications
- Real-time status updates (Applied, Interviewing, Offer, Rejected)
- Search, filter, and sort functionality
- Application statistics dashboard
- Responsive, modern UI with clean layout and visual feedback
- Deployed full-stack application (frontend + backend)

## Tech Stack
Frontend:
- React (Vite)
- CSS

Backend:
- Node.js
- Express
- SQLite
- JWT Authentication
